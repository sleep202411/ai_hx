## shrink 收缩计算
- 溢出空间 300px  让子元素来shrink 分摊
- 分摊的比例？
  flex-shrink：2:1  500:400

  500-300 * 500 需要分摊的比例
  400-300 * 400 需要分摊的比例
- 分摊的权重
  总的权重？
  500*2 + 400*1 = 1400
  1000/1400 = 0.714 71.4%
  400/1400 = 0.286 28.6%

  300 * 0.714 = 214.2
  300 * 0.826 = 85.8

  left 500-214.2 = 285.8
  right 400-85.8 = 314.2

## grow
  600-300-200 = 100 剩余空间
  flex-grow 1:2
  100 * 1/3 = 33.333
  100 * 2/3 = 66.666
  left 300 + 33.333 = 333.333
  right 200 + 66.666 = 266.667
